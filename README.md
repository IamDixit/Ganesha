# Ganesha
This is the backend server to support incoming API requests from web and mobile platform
## Technology
- NodeJS (10.0+)(ES6 standered)
- MongoDB (4.0+) (Mongoose ORM)
- MySQL (Sequalise ORM)
- CI/CD integration
- AWS Lambda and S3 bucket support
